\# 🛒 E-Commerce Platform



Full Stack E-Commerce Platform built with MERN Stack.



\## Tech Stack



\- \*\*Frontend:\*\* React + Vite + Tailwind CSS

\- \*\*Backend:\*\* Node.js + Express.js

\- \*\*Database:\*\* MongoDB

\- \*\*Payment:\*\* Stripe API

\- \*\*Auth:\*\* JWT + OAuth 2.0
